# Steam 游戏库 WordPress 插件

## 功能

本插件注册了一个页面模板，用于展示 Steam 游戏库。

## 使用方法
1. 拉取本仓库所有文件，包括**子模块**:`git clone --recursive`。
2. 上传并启用插件
3. 在`后台`->`Steam游戏库`中填写对应的账户与配置信息<br>![设置界面](assets/image/settingtip.png)
4. 新建页面，页面模板选择`Steam库`
5. 保存页面，前台预览
