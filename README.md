# Steam 游戏库 WordPress 插件

## 功能

本插件注册了一个页面模板，用于展示 Steam 游戏库。

## 使用方法
1. 上传并启用插件
2. 在`后台`->`Steam游戏库`中填写对应的账户与配置信息
    ![设置界面](assets/image/settingtip.png)
3. 新建页面，页面模板选择`Steam库`
4. 保存页面，前台预览
